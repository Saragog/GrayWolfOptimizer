English:

Project for course of ALHE - Heuristic Algorithms.
The goal of this project was to implement heuristic algorithm:
"Gray Wolf Optimizer".
The Project was done in two-person team:
Andrzej Dackiewicz
Mateusz Jarzemski
Project was written in R language on rstudio IDE.

Heuristic Algorithms are used to find good solutions for problems for which
there are not known algorithms that would give ideal solutions.
Grey Wolf Optimizer is an algorithm based on social leadership and behaviour of wolf packs.
